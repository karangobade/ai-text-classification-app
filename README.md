# AI Text Classification App

## About This Project

This is a simple Python project that checks whether a sentence is positive or negative using Artificial Intelligence.

I built this project using PyTorch and Hugging Face Transformers. The application uses a pre-trained DistilBERT model for sentiment analysis.

For example:

- "I love this product!" → Positive
- "This movie was terrible." → Negative

The main purpose of this project is to understand how AI models are downloaded, loaded, and used in real applications.



# Technologies Used

- Python
- PyTorch
- Hugging Face Transformers
- Requests Library
